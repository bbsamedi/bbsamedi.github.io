第1章 绪论

绪论中需要记得东西几乎没有。

1.1 程序设计语言

1.1.1 机器语言

机器语言(machine language)是用二进制代码表示的计算机能直接识别和执行的机器指令的集合。具有计算机直接执行、速度快和效率高等特点。编写的程序都是“0”和“1”表示的，可读性差。可移植性差。

1.1.2 汇编语言

汇编语言(assembly language)采用助记符代表计算机语言的操作。犹豫计算机只能执行机器语言程序，必须将汇编语言程序翻译，完成翻译任务的程序成为汇编器(assembler)。

1.1.3 高级语言

高级语言是比较接近自然语言和数学语言的程序设计语言。高级语言的翻译可用解释和编译两种方式实现：

（1）解释方法，即将高级语言程序中的一条语句单独的转换成机器语言程序，并立即执行，然后进行下一条语句的转换和执行。BASIC语言程序采用结实方式转换成机器语言程序。

（2）编译方法，即将高级语言程序中的所有语句作为一个完整单位转换成机器语言程序，然后执行。C语言程序和C++语言程序都采用编译方式。

1.2 C语言概况

1.2.1 C语言的特点

（1）适应性强，能适应从8位微型机到巨型机的所有机型。

（2）C语言是结构化的程序设计语言，符合现代编程风格要求。

（3）C语言允许直接访问物理地址，能进行位操作，能实现汇编语言的大部分功能，可以直接对硬件进行操作。

（4）生成目标代码质量高，程序执行率高，一般只比汇编程序生成的目标代码效率低10%～20%。

（5）用C语言写的程序可移植性好，基本上不做修改就能用于各种型号的计算机和各种操作系统。

1.3 C语言程序的基本结构

#include<stdio.h>
int main(void)
{
    printf("Hello, the world!\n");
    return 0;
}

“#include<stdio.h>”将文件stdio.h的内容包含在程序中，stdio.h是关于标准输入/输出的头文件（stdio是standard input output的简写，h是head的首字母）

main是函数名，每一个C语言程序必须有且仅有一个main函数。

由一对大括号“{}”括起来的内容是函数体（function body）。

“\n”表示换行，其作用是使光标移到下一行的行首。

/*…….*/是注释信息。若只注释一行，也可以用//……的形式加注释

C语言基本结构总结：

（1）C语言程序是由函数构成的。一个C语言程序至少包含一个main函数，也可以包含一个main函数和若干个其他函数。

（2）C语言程序总是从main函数开始执行。

（3）C语言程序书写格式自由。但一般遵循以下规则来提高可读性：

​ 1⃣️一行内仅写一条语句；2⃣️正反大括号分别各占一行；3⃣️每对大括号上下对齐；4⃣️语句采用缩紧格式，错落有致。

（4）每条语句的最后必须有一个分号，分号是C语言的组成部分。

（5）C语言本身没有输入/输出语句，输入/输出的操作由scanf和printf等函数来完成。

（6）可以用/*……*/或//……在C语言程序中加注释，以增强程序的可读性。

1.4 C语言程序的上机执行过程

1.编辑（edit）

编辑指源程序的输入，对应的文件称为源文件，其扩展名为“.c”。

2.编译（compile）

编译时使用编译器（compiler）将源文件转换为目标文件的过程。目标文件的扩展名为“.obj”。

3.连接（link）

连接是将目标文件和其他分别进行编译生成的目标文件（如果有的话）以及库函数连接生成可执行文件的过程。可执行文件的扩展名为“.exe”。

4.运行

运行时将可执行文件投入运行，以获取程序处理的结果。如果程序运行结果不正确，则回到第一步，重新对程序进行编辑、编译、连接和运行，知道取得预期结果为止。

第2章 标记、类型与表达式

2.1 标记

标记（token）是具有唯一含义的语言的最小单位，分为5种：关键字（keyword）、标识符（indetifier）、常量（constant）、变量（constant）、串字面量（string literal）以及标点符号（punctuator）。

2.1.1 关键字

关键字也称为保留字，所有关键字都有固定的含义，且其含义不可改变。C语言共有44个关键字：

2.1.2 标识符

标识符由小写字母、大写字母、数字、下划线、通用字符名（universal character name）或实现定义的字符（implementation-defined character）构成，且数字不能作为标识符的第一个字符，关键字不能作为标识符。

大写字母和小写字母是有区别的，即所谓大小写敏感。C语言对标识符的最大长度没有具体的限制，VC允许标识符的最大长度为247个字符。

2.1.3 常量

在程序运行过程中，其值不变的量称为常量。

1.整型常量（integer constant）

整数包括正整数、零和负整数。整数常量只包括正整数和零，不包括负整数。

整型常量可分为十进制、八进制和十六进制。

十进制整形常量由0～9组成，且以非零数字开头，或者只有一个0，如123、0。

八进制整型常量由0～7组成，且以0开头，如056、017。

十六进制整型常量由0～9、A～F（或a～f）组成，且以0x或0X开头，如0x1a、0XD5。

2.浮点常量（floating constant）

计算机中的浮点数只能近似地表示值在某个范围之内的有理数和一些特殊值，如NAN（非数值）、+INF（正无穷大）、-INF（负无穷大）等。

浮点常量是非负的浮点数，其十进制书写熊市有以下两种：

（1）小数表示法，由0～9和小数点组成，必须有小数点。如果小数点左边为0，则0可省略；如果小数点右边为0，则0可省略。例如：

123.0 123. 0.0 .0 0.

（2）指数表示法，通常用在计算中遇到的很大或很小的数。例如，60000.0的指数表示法为6E4；E也可以写成e，如0.000035的指针表示法为3.5e-5。指数表示法有两种形式：

第一种形式：十进制整数常量E符号位 十进制整形常量

第二种形式：浮点常数的小数表示法E符号位 十进制整数常量

其中，E也可以写成e，“符号位”即正负号是可选的。

3.枚举常量（enumeration constant）

枚举常量是类型为int的标识符。

4.字符常量（character constant）

C语言采用哪种编码方案是由运行环境确定的。

字符常量分为整数字符常量和宽字符常量。整数字符常量是由一对单引号括起来的一个或多个字符，其类型为int，在内存中占4个字符。

单引号内的字符可以是普通字符，如’?’、’A’、’0’、’ ‘，其中最后一个是空格符。**字符常量参与运算实际是其ASCII码值参与运算。

（1）数字字符0~9的ASCII码值是连续递增的。

（2）大写字母A~Z的ASCII码值是连续递增的。

（3）小写字母a~z的ASCII码值是连续递增的。

（4）大写字母的ASCII码比相应小写字母的ASCII码值小32.

2.1.4 串字面量

串字面量分为三种：字符串字面量、UTF-8串字面量和宽串字面量。

字符串字面量是用一对双引号括起来的零个或多个字符。在翻译的第七个阶段，空字符即’\0’被加到字符串字面量的末尾，然后字符串字面量以数组元素类型为char的数组的形式存储在内存中。

如果字符串字面量中不含’\0’，则该字符字面量又称字符串，即字符串是以空字符’\0’结束的连续字符序列，其中空字符是在翻译的第七个阶段加上去的。

不要将字符常量与字符串相混淆。例：’c’是字符常量，占4个字节；而”c”是字符串，占2个字节。

2.3 变量与常用类型说明符

2.3.1 变量

对象是执行环境中的数据存储区，其内容表示值。变量是有名字的对象。变量声明的简单形式：

类型说明符 变量名1 [，变量名2，变量名3,...,变量名n];

其中，变量名是一个标识符。

常用类型说明符有四种，分别是int、char、float和double。

C语言允许在程序的任何位置声明变量，只要满足先声明后使用的原则即可。

2.3.2 int类型

int类型的数据占4个字节，在VC中，能表示的数据范围是-2147483648~2147483647

在0~2147243647范围内的整数常量，其类型是int。

如果是超出2147483646的数据，会产生一处，属于未定义行为，未定义行为的后果是不可预测的，在程序设计中要避免产生未定义行为。

2.3.3 char类型

char也属于整数类型，char类型的数据占1个字节，在VC中，char默认能表示的数据范围是-128~127。char类型变量通常用于存储单个字符，实际存储的是该字符所对应的ASCII码值，即一个整数。

float和double都属于是舒服点类型，双精度浮点类型double比单精度浮点类型float的精度更高，表示数据的范围更大。在不同的C编译系统中，float类型的数据和double类型的数据所占用的内存大小不同，能表示的数据范围也不同。在VC中，二者的区别

2.4 运算符与表达式

2.4.1 表达式

表达式是由运算符和运算对象构成的序列。表达式具有以下一个或多个功能：

（1）描述一个值的计算。

（2）制定一个对象或一个函数。

（3）产生副作用。

表达式的计算总是先计算运算对象的值，再根据运算符的含义计算表达式的值。

访问异变对象，修改对象或文件，或者调用包含这些操作的函数都是副作用。副作用会改变执行环境的状态。

根据运算符的运算对象的数量，可将运算符分为以下三种:

（1）单目运算符：只有一个运算对象。

（2）双目运算符：有两个运算对象。

（3）三目运算符：有三个运算对象。

2.4.2 乘法类运算符

乘法类运算符有三个，分别是：*、/和%，都是双目运算符。

乘法运算符是*，表达式的值是两个运算对象的乘积。

除法运算符是/，表达式的值是第一个运算对象除以第二个运算对象的商。

模运算符是%，又称求余运算符，要求两个运算对象必须是整数类型，表达式的值是第一个运算对象除以第二个运算对象的余数，其符号与第一个运算对象相同。

2.4.3 加法类运算符

加法类运算符有两个，分别是+和-，都是双目运算符。

如果加法运算符+的两个运算对象都是算数类型，表达式的值是两个运算对象的和。

如果减法运算符-的两个运算对象都是算数类型，表达式的值是第一个运算对象减第二个运算对象的差。

2.4.4 运算符的优先级与结合性

当表达式中含有多个运算符时，要根据运算符的优先级与结合性来理解表达式的含义。优先级高的运算符先绑定运算对象。

结合性分为两种：左结合性和右结合性。优先级相同的运算符按结合性绑定运算对象，左结合性指左边的运算符先绑定运算对象，依次向右进行；右结合性指右边的运算符先绑定运算对象，依次向左进行。

2.4.5 sizeof运算符

sizeof运算符是单目运算符，表达式的值是运算对象所占内存大小（按字节计算），其运算对象是表达式或用小括号括起来的类型名。

char类型、int类型、float类型和double类型在内存中依次占1个字节、4个字节、4个字节和8个字节。

2.4.6 一元加运算符与一元减算符

一元加运算符+是单目运算符，表达式的值是运算对象的值。

一元减运算符-是单目运算符，表达式的值是运算对象的相反数。

如果运算对象是整数类型，要先对运算对象的值进行整数提升，表达式的类型是提升以后的类型。对于常用类型char、int、float和double，只有插入类型的值需要整数提升，要将char类型的值转换为int类型的值。

例：变量a是char类型，在内存中占1个字节，整数提升将a的值转换为int类型的值，因此，表达式+a和-a的类型都是int，在内存中占4个字节。 变量a的类型并没有改变，仍然是char类型。

根据整数常量的定义，7是整数常量，+7和-7都不是整数常量，而是表达式。

2.4.7 常用算数转换

乘法类运算符的对象要执行常用算术转换；如果加法类运算符的两个运算对象都是算法类型，也要执行常用算数转换。对于常用类型char、int、float和double类型，常用转换的含义如下：1）如果其中一个运算对象是double类型，另一个运算对象的值被转换为double类型的值。2）如果以上条件不满足，并且其中一个运算对象是float类型，另一个运算对象的值被转换为float类型的值。3）如果以上两个条件不满足，对两个运算对象进行整数提升，即char类型运算对象的值被转换为int类型的值。

2.4.8 赋值运算符

赋值运算符分为两种：简单赋值运算符和复合赋值运算符，都是双目运算符。1.简单赋值运算符=；2.复合赋值运算符*=、/=、%=、+=和-=。

2.4.9 后缀增1运算符与后缀减1运算符

类型转换运算符是（type-name），属于单目运算符，表达式的值是将运算的值转换为type-name类型得到的值，表达式的类型是type-name

2.4.12 逗号运算符

逗号运算符“，”是双目运算符，表达式的值是有运算对象的值，表达式的类型是有运算对象的类型。

逗号运算符的两个运算对象之间有序点，与左运算对象相关的每一个值计算和副作用完成后，与有运算对象相关的每一个值计算和副作用才开始。

执行序列中某个特定的点被称为序点。

printf(“%d %d”, 10, (20,30)); //输出结果为10 30

以上语句中共有三个逗号， 前两个是分隔符，第三个是逗号运算符。

第3章 顺序结构

3.1 结构化程序设计

为解决问题而采取的方法和步骤，称为算法。

计算机算法可分为两大类：数值运算算法，主要用于求解数值问题；非数值运算算法，主要用于事务管理领域。算法有五个特征。

1）有穷性：一个算法应包含有限的操作步骤而不能是无限的。

2）确定性：算法中每一个步骤应当是确定的，而不能是含糊的、模棱两可的。

3）有效性：算法中每一个步骤应当能有效地执行，并得到确定的结果。

4）有零个或多个输入。

5）有一个或多个输入。

3.2 语句

语句指定要执行的动作，分为六种：表达式语句、复合语句、选择语句、循环语句、标号语句和跳转语句。

3.2.1 表达式语句

在表达式的末尾加上给一个分号构成表达式语句。

有一种特殊的表达式语句，成为空语句（null statement）。空语句中没有表达式，只有一个分号。空语句不执行任何操作。

2.2.2 复合语句

将声明或语句用一对大口号扣起来形成的语句称为复合语句。复合语句又称为块（block）。例如，maiin函数的函数体就是复合语句。

3.3 常用类型数据的输出和输入

3.3.1 数据输入/输出的概念

输入/输出是以计算机为主体而言的。C语言本身不提供输入、输出操作的语句。输入、输出操作湿透透弄故宫调用库函数来实现的。

3.3.2 常用类型数据和字符串的输出

调用库函数printf的一般形式:

​ printf(格式控制字符串,表达式1,…,表达式n);

其功能：根据格式控制字符串，输出表达式1到表达式n的值。

格式控制字符串由普通字符（包括转义字符，但不含%）和转换说明组成。普通字符在输出时原样输出，而转换说明用于控控制表达式1到表达式n的输出形式。

转换说明是%开头的一个或多个字符。转换说明符是%之后欧必须有的字符。

转换说明符 含义
d 以有符号十进制形式输出int类型的值
c 输出int类型的值所对应的字符（根据ASCII码表）
f 以十进制小数像是输出double类型的值
s 输出字符串(空字符’\0’不输出)
u 无符号 十进制数
o 无符号 八进制数
x 使用十六进制0f的无符号十六进制数
X 使用十六进制0F的无符号十六进制数
p 指针
lf double类型浮点数 十进制计数法
% 输出一个%

调用库函数printf输出char类型的值时，先对char类型的值进行整数提升，char类型的值被提升为int类型。因此，可用%d或%c输出char类型的值。

调用库函数printf输出float类型的值时，float类型的值先被转换为double类型。因此。可用%f输出float类型的值。

在%和转换说明符之间加正整数，可控制数据局输出的宽度。默认情况下，%f小数半部分输出6位，在%和f之间加“.正整数”，可控制小数部分的输出位数。

3.3.3 常用类型数据的输入

调用库函数scanf的一般形式：

​ scanf(格式控制字符串，变量1的地址，变量2的地址，…，变量n的地址);

其功能是：根据格式控制字符串，读取键盘输入，并存储到相应的变量中。

转换说明符 含义
d 读取有符号十进制整数并存储到int类型的变量中
c 读取一个字符并存储到char类型的变量中
f 读取有符号浮点数并存储到float类型的变量中

调用库函数scanf读取多个数据（字符数据除外）时，输入的数据之间应该用空格符、tab符或回车符作为分隔符。

当使用%c读取一个字符并存储到char类型的变量中时，空格符、回车符等也会被作为一个字符被读取。

3.4 库函数printf详解

对于库函数printf而言，还有更为复杂的使用方法。格式控制字符串的一般形式为：

%[标志][输出最小噢宽度[.精度][长度修饰符]转换说明符

1.标志

标志字符为-、+、空格和#四种，其含义如表

标志 含义
- 结果左对齐，右边填空格
+ 输出正数时，在数字前加上+号
空格 输出值为正时，在数字前加上空格，为负时冠以负号
# 对c、s、d、u类型无影响：对o类型，在输出时加前缀o:
对x类型，在输出时加前缀0x：对e、g、f类型，当结果有小数时才给出小数点

2.输出最小宽度

用十进制整数来标识输出的最小位数。若实际位数多于定义的宽度，则按实际位数输出；若实际位数少于定义的 宽度，则补以空格或0.

3.精度

精度格式符以“.”开头，后跟十进制整数。他所表示的意思是：如果过输出实数，则表示小数的位数；如果输出的是字符串，则标识输出字符的个数。

4.长度修饰符

长度修饰符有h和l两种，当输出整数时，%hd表示按short型输出，%ld表示按long int类型输出。

3.5 库函数scanf详解

对于库函数scanf而言，也有更为复杂的使用方式。。格式控制字符串的一般形式为：

%[*][输入数据宽度][长度修饰符]转换说明符

1.*

*是抑制符，读取的数据不会存储到相应的变量中，即跳过该输入值。

2.输入数据宽度

指定输入数据的宽度，遇到空格或者不可转换的字符则结束。

3.长度修饰符

长度修饰符有h和l两种，字符l用于类型d、u、o、x前，指定输入数据为long型。

库函数scanf_s和库函数scanf的功能相同，但调用库函数scanf_s更安全。

第4章 选择结构

4.1 关系运算符和相等类运算符

4.1.1 关系运算符

关系运算符有四个，分别死<、>、<=、>=，都是双目运算符。

关系表达式的类型是int。如果指定的关系为真，关系表达式的值是1；如果指定的关系为假，关系表达式的值是0.

4.1.2 相等类运算符

相等类运算符有两个，分别是==和!=，都是双目运算符。

相等类表达式的类型是int。如果指定的关系为真，相等类表达式的值是1；相反则为0。

4.2 逻辑运算符

4.2.1 逻辑非运算符

逻辑非运算符是！，属于单目运算符。逻辑非运算符和运算对象构成的表达式的类型为int。如果运算对象的值不等于0，表达式的值是0；如果运算对象的值等于0，表达式的值是1。

4.2.2 逻辑与运算符

逻辑与运算符是&&，属于双目运算符。逻辑与运算符和运算对象构成的表达式的类型是int。如果两个运算对象的值都不等于0，表达式的值是1；否则，表达式的值是0。即任何一个运算对象的值等于0，表达式的值是0。

4.2.3 逻辑或运算符

逻辑或运算符是||，属于双目运算符。逻辑或运算符和运算对象构成的表达式的类型是int。如果任何一个运算对象的值不等于0，表达式的值是1；否则，表达式的值是0。即只有两个运算对象的值都等于0，表达式的值才是0.

4.3 if语句

4.3.1 第一种形式

        语句

其功能是：如果表达式的值不等于0，则执行语句1；否则，执行语句2。

4.4 条件运算符

条件运算符为“? :”，是C语言中唯一的三目运算符。条件表达式的一般形式：

运算对象1 ? 运算对象2 : 运算对象3

先计算1，在“运算对象1的计算”与“运算对象2或运算对象3的计算”之间有序点，2和3中只有一个会被计算。

如果运算对象1的值不等于0，计算运算对象2，条件表达式的值是运算对象2的值；否则，计算运算对象3，条件表达式的值是运算对象3的值。例如：0?3:5的值是5。

在表达式a?b:c?d:e中，有两个条件运算符，按结合性绑定运算对象，条件运算符是右结合性，所以右边的条件运算符先绑定运算对象，表达式的含义为a?b:(c?d:e)。

switch语句的功能如下：

（1）switch后面表达式（即控制表达式）的类型只能是整数类型，而且会对控制表达式进行整数提升。

（2）每个case后面整数常量表达式会被转换为控制表达式提升以后的类型，转换后的值必须互相不相同。

如果某个case后面整数常量表达式被转换后的值等于被提升的控制表达式的值

否则

​ 如果有default项

​ 执行default后面的语句

​ 否则

​ 没有语句被执行

（3）执行完某个case后面的语句后，并不会退出switch语句，而是一直执行下去，直到switch语句结束。

（4）default项通常放在最后，也可以放在任何位置，其功能是相同的。唯一的区别是：放在最后，不需要加break语句；放在其他位置，通常要加break语句。

第5章 循环结构

5.1 for语句

for语句的一般形式：

for (表达式1;表达式2;表达式3)
        循环体

for语句的功能如下：

（1）计算表达式1。

（2）计算表达式2。如果表达式2的值不等于0，进入步骤（3）；否则，退出for语句。

（3）执行循环体。

（4）计算表达式3，返回步骤（2）。

for语句中的“表达式1、2、3”都是可选的。如：

int i;                                int i;                                inti;
for(i=1;i<10;i++)            for(;i<10;i++)                for(i=1;i<10;)
    printf("?");                    printf("?");                {
                                                                                            printf("?");
                                                                                            i++;
                                                                                        }

虽然“表达式1、3”可以省略，但关键字for后面口号内的两个分号不能省略。如果省略了“表达式2”，省略的“表达式2”会被一个非零常量替换，相当于“表达式2”永远不等于0，导致“死循环”。

C语言允许使用另一个形式的for语句，即在“表达式1”的位置声明循环变量：

            语句

5.2 while语句

while语句的一般形式：

while (表达式)
        循环体

while语句的功能如下：

（1）计算表达式。如果表达式的值不等于0，进入步骤（2）；否则，退出while语句。

（2）执行循环体，返回步骤（1）。

注意：

虽然for语句能实现的功能，while语句也能实现，但是for语句通常用于循环次数在执行循环体之前已确定的情形，而while语句通常用于循环次数在执行循环体之前不能确定的情形。

以下两while语句是等价的，右侧的while语句使用更多一些。

while (x!=0)                                while (x)
        语句                                                语句

5.3 do-while语句

dou-while语句的一般形式：

do
        循环体
while (表达式);

do-while语句的功能如下：

（1）执行循环体。

（2）计算表达式。如果表达式的值不等于0，进入步骤（1）；否则，退出do-while语句。

注意：

for语句和while语句的循环体最少执行0次，而do-while语句的循环体最少执行1次。do-while语句与while语句类似，通常用于循环次数在执行循环体之前不能确定的情形。

5.4 标号语句和跳转语句

5.4.1 标号语句

标号语句有三种形式：

​ 标识符：语句

​ case整数常量表达式：语句

​ default：语句

只能在switch语句中使用case标号或default标号。

在任何语句之前加上前缀“标识符:”构成标号语句。

5.4.2 跳转语句

跳转语句分为四种：goto语句、continue语句、break语句和return语句。

1. goto语句

goto语句的一般形式：

goto 标号;

其中，标号是某个标号语句中冒号前面的标识符。goto语句使控制无条件跳转到标号所对应的标号语句。

2. continue语句

continue语句的一般形式：

continue;

只能在循环体中使用continue语句。continue语句使控制流程跳转到包含他的最内层循环体的最后位置，即跳过包含他的最内层循环体中他最后的语句。

3. break语句

break语句的一般形式：

break;

只能在switch语句或循环体中使用break语句。break语句使包含它的最内层switch语句或最内层循环语句终止执行。

4. return语句

return语句有两种形式：

return 表达式;
return;

return语句终止当前函数的执行，并将控制返回给该函数的主调函数。

5.5 嵌套的循环语句

循环语句（外循环）的循环体中包含另一个循环语句（内循环），称为嵌套的循环语句。

例题：输出所有的水仙花数。水仙花数是一个三位正整数，每位数字的三次方之和等于该数本身。

#include<stdio.h>
int main(void)
{
    int i, j, k, m, n;
    for(i=1;i<=9;i++)
        for(j=0;j<=9;j++)
            for(k=0;k<=9;k++)
            {
                m = i * 100 + j * 10 + k;
                n = i * i * i + j * j * j + k * k * k;
                if (m == n)
                        printf("%5d",m);
            }
    printf("\n");
    return 0;
}

第6章 数组类型

6.1 一维数组

6.1.1 数组声明

一维数组声明的简单形式：

类型说明符 数组名[整数类型表达式]

其中，类型说明符是数组元素的类型，数组名是一个标识符，符号“[”和“]”内的整数类型表达式表示数组元素的个数，即数组的长度。如果整数类型表达式是常量表达式，其值必须大于0。

数组名是数组类型的变量，是不可改变的左值。数组名作为运算符sizeof的运算对象，运算结果是全部数组元素占内存大小的总和（按字节计算）。

6.1.2 引用数组元素

数组元素的一般形式：

数组名[整数类型表达式];                    //这里的[ ]是下标运算符

整数类型表达式也称为下标，下标的取值范围是0～数组长度-1。

一维数组名的值是第0个数组元素的地址。

数组不能整体输入、输出和赋值，只能对数组元素进行输入、输出和赋值。

6.1.3 初始化数组

在数组声明时为数组赋值，称为初始化。

如果初始值的个数少于数组长度，剩余的数组元素被初始化为零值。

在数组声明时为数组赋值，可以省略数组长度，此时数组长度即为初始值的个数。

只能在数组声明时为数组赋值，在其他位置只能为数组元素赋值，不能为数组赋值。

6.2 二维数组

6.2.1 数组声明

多为数组中常用的是二维数组。二维数组声明的简单形式：

类型说明符 数组名[行数][列数];

其中，类型说明符是数组元素的类型；数组名是一个标识符；行数和列数是整数类型表达式，数组元素的个数是行数与列数的乘积。如果表示行数或列数的表达式是整数常量表达式，其值必须大于0。

6.2.2 引用数组元素

数组元素的一般形式：

数组名[行号][列号];                    //这里的[]是下标运算符

行号和列号是整数类型表达式，行号的取值范围是0～行数-1，列号的取值范围是0～列数-1。

6.2.3 初始化数组

int a[3][5] = {{1,3,5,7,9},{2,4,6,8,10},{11,12,13,14,15}};            //1
int a[3][5] = {1,3,5,7,9,2,4,6,8,10,11,12,13,14,15};                        //2

两种初始化的效果相同，推荐使用方式1初始化。

6.3 数组与字符串

6.3.1 使用数组元素类型为char的数组存储字符串

数组元素类型为char的数组可存储多个字符，如果最后一个字符是空字符’\0’，则可将这多个字符理解为字符串，即字符串可存储在数组元素类型为char的数组中。

有三种初始化数组的方法可将字符串存储到数组元素类型为char的数组中：

char s[5] = {'b', 'l', 'u', 'e', '\0'};
char s[5] = {"blue"};
char s[5] = "blue";

如果数组元素类型为char的数组中存储的不是字符串，则不能用%s输出。

说明

转换说明%s读取键盘输入的字符，并存储到数组中，遇到空格符、Tab符或回车符停止读取，并将’\0’存储到最后一个字符所在数组元素的下一个数组元素中，空格符、Tab符或回车符被丢弃；

如果输入的字符串中含有空格符，转换说明%s遇到空格符停止读取。

6.3.2 常用字符串处理库函数

（1）库函数puts（需包含头文件stdio.h）

调用库函数puts的一般形式：

puts(字符串);
或
puts(数组元素类型为char且存储字符串的数组名);

库函数puts的功能：输出字符串，并在字符串的末尾输入’\n’，不输出’\0’。

（2）库函数gets（需包含头文件stdio.h）

调用库函数gets的一般形式：

gets(数组元素类型为char的数组名);

库函数gets的功能：读取键盘输入的字符，并存储到数组中，遇到回车符停止读取，并将’\0’存储到最后一个字符所在数组元素的下一个数组元素中，回车符被丢弃。既可以调用库函数gets输入含有空格符的字符串。

（3）库函数strlen（需包含头文件string.h）

调用库函数strlen的一般形式：

strlen(字符串);
或
strlen(数组元素类型为char且存储字符串的数组名);

库函数strlen的功能：计算并返回字符串的长度，不包含’\0’。

（4）库函数strcpy（需包含头文件string.h）

调用库函数strcpy的一般形式：

strcpy(数组元素类型为char的数组名,字符串);
strcpy(数组元素类型为char的数组名,数组元素类型为char且存储字符串的数组名);

库函数strcpy的功能：复制字符串到第一个数组中，包括’\0’。

（5）库函数strcmp（需包含头文件string.h）

调用库函数strcmp的一般形式：

strcmp(字符串1,字符串2);
strcmp(数组元素类型为char且存储字符串的数组名,字符串);
strcmp(字符串,数组元素类型为char且存储字符出啊你的数组名);
strcmp(数组元素类型为char且存储字符串的数组名1,数组元素类型为char且存储字符串的数组名2);

库函数strcmp的功能：比较两个字符串的大小，如果第一个字符串大于第二个字符串，返回一个正整数；如果两个字符串相等，返回0；如果第一个字符串小于第二个字符串，返回一个负整数。

两个字符串比较大小的规则：从左到右按ASCII码值逐个字符比较大小(包括’\0’)，如果全部字符相等，则两个字符串相等，返回0；如果有一个不相等的字符，以第一个不相等字符的比较结果作为两个字符串的比较结果，返回第一对不相等字符的ASCII码值的差。

第7章 指针类型

7.1 指针与指针变量

7.1.1 指针的概念

内存中的每个存储单元（即字节）有一个编号 ，称为地址。地址从0开始依次编号。有梁总方式可以访问内存中的存储单元：直接访问方式和间接访问方式。

1.直接访问方式

按变量名存取变量值的方式称为“直接访问”方式。

如果在程序中声明了变量，编译系统就会为变量按其类型分配相应的存储单元，并将变量名与其对应单元的地址建立联系。例如：

int i;
char ch;

一个字节一个地址编号，即一个存储单元。

2.间接访问方式

间接访问方式是将变量的地址存储到另一个变量中，指针类型的变量（简称指针变量）用于存储地址。例如：

int x,*p;
x=10;
p=&x;        //将变量x的地址存储到指针变量p中

指针变量p中存储的是变量x的地址，若要访问变量x，可先通过p找到x的地址3000，再根据地址3000找到x

7.1.2 指针变量的声明

指针变量声明的简单形式：

类型说明符 *变量名1 [, *变量名2, ..., *变量名n];

其中，变量名是一个标识符。

在声明指针变量时要注意：

（1）“变量名”前面的“*”是一个标志，表明其后的变量是指针类型，该变量只能存储地址。

（2）“类型说明符”表示该指针变量所指向的对象的类型。

（3）指针中只能存入由系统分配的地址。

7.1.3 取地址运算符和间接寻址运算符

取地址运算符是&，表达式的值是运算对象的地址。

间接寻址运算符是*，其运算对象必须是指针变量，表达式的值是指针变量所指对象的左值。例如：

int x,*P;
p = &X;            //p指向x
*p =10;            //相当于x=10；

在使用取地址运算符和间接寻址运算符时要注意以下几点：

（1）“*”号用在不同的地方有不同的含义：①声明指针变量的标志；②间接寻址运算符；③乘法运算符。例如：

int x;
int *p;                //*表明p是指针变量
p = &x;
scanf("%d", p );
*p = 3 * x;            //第一个*号是间接寻址运算符，第二个*是乘法运算符

（2）&x的结果是x；如果p是指针变量，&\p的结果是p。

（3）取地址运算符的运算对象必须是左值或函数指示符，&x不能出现在赋值号的左边。

（4）未赋值的指针变量不能作为间接寻址运算符的运算对象。如果指针变量p未被赋值，那么试图使用或改变p所指对象的值会导致未定义的行为。

空指针是NULL，如果指针变量的值是NULL，表明该指针变量不能指向任何对象。

7.2 指针与一维数组

7.2.1 使指针指向数组元素

使指针指向数组元素与使指针指向变量的方法相同。例如：

int a[4] = {1, 3, 5, 7};            //数组元素类型为int长度为4的一维数组a
int *p;                                //指向int类型对象的指针变量p
p = a;                                //相当于p = &a[0];

7.2.2 下标运算符

下标运算符是[]，一个运算对象的类型是指针类型，另一个运算对象的类型是整数类型，表达式E1[E2]的值是((E1)+(E2))，是间接寻址运算符。

7.3 指针与二维数组

7.3.1 二维数组中的地址

在C语言中，二维数组是按行排列的，允许将一个二维数组分解成多个一维数组来处理。因此可将二维数组理解为特殊的一维数组，其每个数组元素又是一个一位数组。

每行是一个一维数组，只要能确定每行的首地址即行首地址，就能知道该行的所有元素的地址。

a[i]、a+i、*(a+i)、&a[i][0]都表示行首地址。

*(a[i]+j)、*(*(a+i)+j)、(*(a_i))[j]、a[i][j]都表示i行j列元素。

7.3.2 引用数组元素

利用指针访问二维数组有两种方式：指向数组元素的指针和行指针。

1.指向数组元素的指针

指向二维数组元素的之后在呢变量的声明与普通指针变量的声明相同。

例：

#include<stdio.h>
int main(void)
{
    int a[3][4]={{1,9,5,13},{23,8,49,2},{30,15,20,6}};
    int i,j,*p,max,k,t;
    p=a[0];            //p为0行首地址，也可写作p=*a或p=*(a+0)
    max=*p;            //max赋值为数组首元素
    k=t=0;            //k为行下标，t为列下标
    for(i=0;i<3;i++)
        for(j=0;j<4;j++,p++)
            if(max<*p)
                max=*p,k=i,t=j;
    printf("max=%d, k=%d, t=%d\n",max,k,t);
    return 0;
}
2.行指针或指向一维数组（二维数组的一行）的指针

上面的例题中的指针变量p只想的是数组元素，即int型对象，p值加1所指向的元素是原来p指向元素的下一个元素。例如，p指向a[0][0]，p+1指向a[0][1]。也可以使用另一种方法，使p不指向int型对象，而是指向一维数组（二维数组的一行）。这里的指针p被称为行指针。

行指针生命的简单形式：

类型说明符 (*指针变量名)[元素个数];

7.4 指针与字符串

可以使用指向char类型对象的指针变量指向字符串的首字符，再通过该指针变量访问字符串。

7.5 指针数组

如果数组元素的类型是指针类型，这样的数组称为指针数组。

第8章 函数

8.1 概述

函数是完成特定功能的程序段。C语言程序由一个或多个函数组成们必须有且仅有一个main函数。程序启动时调用的函数是main函数，即main函数是程序执行的起点。

函数分为库函数和用户定义函数：

（1）库函数：库函数由编译器提供，只需包含相应的头文件，即可在程序中直接调用。

（2）用户定义函数：程序员根据需要编写的函数是用户定义函数。包含用户定义函数的C语言程序通常分为四个部分：①包含头文件；②1n个函数声明；③main函数；④1n个函数定义。

函数定义是平行的，彼此相互独立，不能嵌套定义。将函数output的函数定义放到main函数的函数体内，是错误的。

如果函数A调用函数B，则称函数A为主调函数，称函数B为被调函数。

8.2 函数定义

函数定义的简单形式：

类型说明符 函数名(形式参数及其类型列表) 复合语句

（1）函数头。“类型说明符 函数名（形式参数及其类型列表）”一起成为函数头。

（2）函数体。复合语句即函数体，函数体实现函数的功能。

（3）类型说明符。类型说明符是函数的返回类型，常见的返回类型有基本类型、枚举类型、空类型、指针类型、结构体类型、共同体类型。函数的返回类型不能是数组类型或函数类型。

如果函数的返回类型是空类型，表明函数没有返回值，函数体中可以没有return语句，也可以没有return语句，也可以含有多条return语句，每条return语句的形式如下：

return;

“return;”终止当前函数（即被调函数）的执行，并将控制返回给主调函数。

如果函数的返回类型是除空类型以外的其他类型，表名函数有且仅有一个返回值（返回值得类型是函数的返回类型），函数体中至少含有一条return语句，每条return语句的形式如下：

return 表达式;

“return 表达式;”终止当前函数（即被调函数）的执行，出来将控制返回给主调函数，return后面表达式的值被返回给主调函数，作为函数调用表达式的值。如果return后面表达式的类型与函数的返回类型不同，则return后面表达式的值被转换为函数的返回类型。

（4）函数名。函数名是一个标识符，应该尽量使用能表明定函数功能的函数名。

（5）形式参数。函数定义时，函数名后面括号内的参数称为形式参数。形参必须是可修改的左值，每个形参是前面必须有类型说明符。
